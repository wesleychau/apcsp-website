Change Directory (cd)

This command allows you to change the directory you are looking at. If you type “cd” without any directory name, you are taken to your home directory. Otherwise type to change to a different directory.

Example: cd desktop # takes you to desktop directory

List Files (ls)

This command allows you to list all the files/directories that are located under your current directory. It also is color coded based of whether it is a file, directory, or link.

Example: ls # lists everything under your current directory

Manual (man)

This command allows you to get all the information about a certain command.

Example: man ping # lists all the options associated with the ping command

Ping (ping)

This command allows you to test the connectivity between two nodes. It can return information about the number of packets transmitted and whether the machine is alive. It also has a multitude of options that allow you to ping a specific number of times or only return certain information.

Example: ping rpi14 # returns the time it takes for packets to transmit between the two nodes and if rpi14 is alive

Make Directory (mkdir)

This command allows you to create a new directory under your current directory and then later add files, subdirectories, or links into the new directory you have created.

Example: mkdir website # creates a new directory named “website” under your current directory

Move File (mv)

This command allows you to move a file from a current directory to a new directory. It also has many options including backing up the file or only moving it after an update.

Example: mv index.html website # moves “index.html” file to “website” directory

Change Mode (chmod)

This command allows you to change the file mode bits of each given file according to mode. Often used to make a file executable or change preferences.

Example: chmod -x index.html # makes the “index.html” file executable

Remove (rm)

This command allows you to remove files, directories, and links from file systems.

Example: rm index.html # removes the “index.html” file from the file system

Remove Directory (rmdir)

This command allows you to remove a directory that is empty. It will not remove a directory that contains files.

Example: rmdir website # removes the empty directory “website”

Concatenate Files (cat)

This command allows you to view the contents of a file. It also has other uses such as the creation of single or multiple files or redirecting files which is based off the different options associated with this command

Example: cat index.html website # displays the contents of the “index.html” files

Nano Editor (nano)

This command allows you to create a new file or edit a file that is already in existence. It opens the GNU nano editor which is a popular line text editor. This also allows you to save a file in multiple locations, the options which are given to you after you ctrl^x to exit.

Example: nano index.html # creates a file named “index.html” and allows you to edit that file
